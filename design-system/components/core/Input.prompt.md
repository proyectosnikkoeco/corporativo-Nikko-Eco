Campo de texto. Label en Geist Mono uppercase, caja con borde fino tinta, radio casi recto. Hint / error van debajo.

```jsx
<Input label="Email" type="email" placeholder="alguien@empresa.com" hint="No usamos tu email para nada más" />
<Input label="Teléfono" type="tel" error="Formato inválido" />
```
