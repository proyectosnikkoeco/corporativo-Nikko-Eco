# Brand assets

Aquí van los recursos visuales definitivos de Nikko Eco (generados con
Claude Design):

- `logo.svg` — logo principal
- `logo-blanco.svg` — versión invertida (fondos oscuros)
- `isotipo.svg` — símbolo aislado
- `og-image.png` — imagen Open Graph (1200×630)
- `fonts/` — fuentes custom si no son de Google Fonts

Los archivos `.svg` actuales son **placeholders**. Reemplazar cuando
lleguen los definitivos. Ver `docs/BRAND.md` para la guía completa.
